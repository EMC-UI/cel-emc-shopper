Cart

Description

The Cart component is a local storage driven "bucket" of things. With it's api, you can add/remove items from the bucket. Each item in the bucket will be persisted to the browser's local storage across session. Once the user is finished with their bucket, this component allows the developer to recieve a checkout event which will supply the items that the user has checked out.

Example uses of the header component

<cart></cart>

<cart [itemTemplate]="{{ item.name }}"
       [summaryTemplate]="{{ items.quantity }} {{ items.total }}"
       (checkout)="checkoutFunction(items)">
</cart>

Inputs

Property Type Details
itemTemplate string Default template is provided with basic item.name displayed.
summaryTemplate string Default is no summary template. items object provided for your template
checkout event Called when user is ready to 'checkout'. first argument is array of items

Outputs

event: checkout - fires with one argument that is an array of items in the cart

Usage

as element:
<cart>
</cart>
